# Synthetix AI Video Production Suite

## Overview
A high-performance web architecture for AI-orchestrated video production. This dashboard facilitates a 6-phase workflow from reverse engineering reference content to final audio synthesis.

## File Map
- `index.html`: Dashboard structure with Tailwind integration.
- `style.css`: Custom animations and Glassmorphism styling.
- `app.js`: State management and AI pipeline simulation logic.

## Technical Specifications
- **Phase 1 & 2 (Logic)**: Uses Claude 4.6 for structural analysis and scriptwriting.
- **Phase 3 (Visuals)**: Uses Nano Banana 2 for image generation prompts.
- **Phase 4 (Motion)**: Uses Veo 3 for image-to-video synthesis.
- **Phase 5 (Voice)**: Uses ElevenLabs v3 for emotive narration.
- **Phase 6 (SFX)**: Uses Seedance for temporal sound design.

## Deployment
1. Ensure all files are in the same directory.
2. Open `index.html` in a modern browser.
3. Enter a URL and click "Analyze" to trigger the sequence.
